This directory contains the minimal files necessary to run the executable.  It does not contain any source code or any of the SDKs.  Thus, this is what would need to be installed on a user's computer to run the program, without the capability to modify or further develop the application.

== Executable ==
   NxTutorial104.exe

== Libraries ==
(PhysX Libraries)
   NxCharacter.dll
   NxCooking.dll
   NxOgre.dll

(OGRE Libraries)
   OgreMain.dll
   OIS.dll
   Plugin_ParticleFX.dll
   RenderSystem_Direct3D9.dll
   RenderSystem_GL.dll

== Configuration Files ==
   config.yaml
